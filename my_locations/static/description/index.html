Load locations of a country

This app allows load to database the location data of a defined country from the GeoNames.

It makes a new field on contact address that allows an auto fill of: zip, city, country, and state.

For more information about locations you can visit the offical website GeoNames.

To import locations of a country, you can do it in: Contacts -> Configuration -> Import Locations by Country

Technical Features

Characteristics:

  • Downloads and loads locations of a defined country.
  • Makes list of locations.
  • Makes list of municipalities or communities.
  • It can be used on Enterprise and Community modes for Odoo 12.
  • Languages: English, Spanish.

Python dependencies classes:

  • requests
  • tempfile
  • io
  • zipfile

Configuration

In the odoo.conf file, you can add two parameters such as:

  • limit_time_cpu = 700
  • limit_time_real = 2048

These values increase more time and hardware resource to download and load of the locations to database. The quantity of the values are an example.

Check the parameter in: Settings -> Technical -> Parameters -> Systems Parameter

Add a key name and its value:

  • Key name: geoname.url
  • Value: http://download.geonames.org/export/zip
